Technical companion to §Mechanism of the Backfire Effect

This report documents (i) what was computed and how — every metric in the mechanism analysis, its definition, and the code path that produced it — and (ii) the measured values together with the exact files that hold them. The entire analysis is in a single notebook, backfire-mechanism-hk.ipynb, and uses only on-disk simulation output (no new simulations). An earlier variant with an Erdős–Rényi contrast (backfire-mechanism-analysis.ipynb) is superseded: the camp-averaged cRate did not differ between topologies, making that contrast uninformative, so the final analysis is single-condition and relies on a seed-level dose-response for identification.

The chain under test (paper §Appearance of the Backfire Effect). manipulation → (1) opposing followers unfollow the hub → (2,3) hub loses its bridging role / connectivity drops → (4) opposing camp forms an echo chamber, concentrated in the hub's former followers → (5) echo chamber raises opposing posting → opposing camp outgrows targeted camp = backfire.

2. What each quantity is, and how it is computed

Outcome — backfire magnitude b

Per-trial log-response ratio (paper Eq. for backfire), from post_count_100step.csv:

b = log( v_opp^post / v_opp^pre ) - log( v_tgt^post / v_tgt^pre )
pre window  t in [19000, 20000]      post window  t in [39000, 40000]
opp = far_opposite + near_opposite   tgt = near_target + far_target  (posts/step)

Opinion classes are direction-relative (relative to the manipulation direction), allowing both directions to be pooled. b>0 = backfire.

(1) Selective unfollowing

  • Unfollows by class (unfollow_by_class.csv): count of followers in each opinion class that drop the hub, split into pre (0–20000) and post (20000–40000) windows. Camp = sum of its two classes.
  • Follower composition (follower_composition_5000step.csv): fraction of the hub's in-neighbours in each class, per 5000-step snapshot.

(2,3) Bridging role

  • betweenness_gc (target_hub_metrics_5000step.csv): betweenness centrality of the hub on the undirected giant component, normalized; the hub is tracked by its step-20000 identity across all snapshots (so the pre-onset trajectory is defined). Directed full-graph betweenness is ≈0 because the hub is a near-pure in-degree sink, hence the giant-component undirected version.
  • target_is_ap (connectivity_5000step.csv): fraction of trials in which the hub is an articulation point of the giant component (its removal disconnects the component).
  • lambda2_G: algebraic connectivity (Fiedler value) of the undirected giant component; delta_lambda2 is the drop when the hub is removed.

(4) Echo chamber

  • cRate (crate_100step.csv): the fraction of an agent's feed posts within ±0.2 (Const.MINIMUM_BC) of its own opinion, averaged within each direction-relative class — the simulation's direct echo-chamber exposure metric.
  • Structural cRate proxy (follower_crate_proxy_5000step.csv): for each agent, the postProb-weighted fraction of its followees whose opinion is within ±0.2 of its own. Agents are split by is_follower — whether they followed the hub at step 20000 (fixed, so unfollowers are still tracked). This isolates the hub's former followers from non-followers within the opposing camp.
  • modularity (nw_stats_5000step.csv): directed Leicht–Newman Q, communities via Louvain on the undirected projection.

(5) Activity & outcome

  • Posting-prob increase (posting_prob_increase_by_class.csv): fraction of agents per class whose mean postProb is higher in [35000,40000] than in [15000,20000].
  • Post share (post_count_100step.csv): camp post count divided by total, per step.

Identification — dose-response

Per-trial Spearman correlation of b against two upstream intensities: opposing-camp unfollows in the post window, and the pre→post change in aggregate opposing-camp cRate. Each correlation uses its own maximal sample (they are not co-dropped).

Honest reading. The upstream cause (unfollowing) is significantly graded with the backfire; the aggregate cRate change is not. This is coherent with link (4): at this high-stubbornness baseline the camp-averaged exposure is a saturated, noisy mediator, while the bridge-loss driver and the follower-level echo effect (0.32→0.74 vs. a flat 0.82 for non-followers) carry the mechanism. The cRate dose-response is limited to n=64 because 36 trials' feed-exposure logging stops at step 20000.

5. Caveats

  • Single condition (HK, μ=0.8); generalisation across networks/parameters is the sensitivity analysis, not this section.
  • The mechanism's specificity is visible at the follower level and in the articulation-point signal, not in camp-averaged cRate.
  • Connectivity and betweenness use the undirected giant component; the original directed full-graph betweenness is retained but unused.
  • Dose-response is observational (cross-seed), not an interventional ablation: it shows the pathway is graded, not that it is the sole cause. An ablation (disabling unfollowing) is noted as future work.
